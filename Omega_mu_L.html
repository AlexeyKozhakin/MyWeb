Анализ функции \( F(p, n, \mu, L, b, M, \phi) \)

Введение

Эту функцию я получил от своего друга — теоретического физика. На данный момент я не глубоко погружен в предметную область, но могу сказать, что она связана с физикой высоких энергий, квантовой статистикой и теорией поля и другими смежными направлениями. Моя задача заключалась в вычислении интеграла от положительной области этой функции. Для упрощения постановки задачи я немного преобразовал исходную формулу. Данная задача представляет интерес, поскольку ее исследование затрагивает аспекты анализа и теоретической физики. Она может быть полезна преподавателям, составителям олимпиад и задачников, а также предложена учащимся и студентам для изучения. В дальнейшем я подробно опишу функцию и формулировку задачи.

Постановка задачи

Глобальная подзадача это взять интеграл от положительной области функции \[\int_{-\infty}^{\infty}\sum_{n=-\infty}^{\infty}F(p, n, \mu, L, b, M, \phi)dp\] Задача сводится к исследованию функции \[ F(p, n, \mu, L, b, M, \phi) = \mu - \sqrt{(\sqrt{M^2+p^2}-b)^2+\frac{4\pi^2}{L^2}(n+\phi)^2} \tag{1} \label{eq:main} \]

  • \(p\) — аргумент функции, поскольку функция симметрична можно исследовать только область \(p>0\)
  • \(n\) — целочисленный параметр
  • \(\mu, L, M\) — положительные параметры \(\in [0, \infty)\)
  • \(b\) — положительные параметры \(\in [-\infty, \infty)\)
  • \(\phi\) — может принимать значение 0 (остальные значения добавлю позже)

    • В исследовании данной функции нужно решить две задачи:

    1. \(N_{max}(\mu, L, b, M, \phi)\) — найти максимальное значение \(n\), при котором максимум функции \(F_{max} > 0\).
    2. \(\{p \mid F(p) = 0\}\) — найти все нули функции.

    Формула для \(N_{max}\)

    Сначала необходимо найти максимум функции \(F_{max}\). Один из известных способов заключается в определении экстремумов с помощью производной. Для этого нужно: Найти первую производную функции. Приравнять её к нулю \[\frac{\partial F(p, n, \mu, L, b, M, \phi)}{\partial p} = 0 \], чтобы найти точки экстремума. В этих точках функция будет достигать локального максимума или минимума. Для выполнения этого шага удобно воспользоваться онлайн-системами для символьного дифференцирования. Одним из таких инструментов является сайт MathDF, который предоставляет быстрый и точный способ нахождения производной. На данном сайте я получил производную, но для уверенности в её правильности желательно проверить результат самостоятельно. \[\frac{\partial F(p, n, \mu, L, b, M, \phi)}{\partial p} = -\dfrac{p\,\left(\sqrt{{p}^{2}+{M}^{2}}-b\right)}{\sqrt{{p}^{2}+{M}^{2}}\,\sqrt{\left({\sqrt{{p}^{2}+{M}^{2}}-b}\right)^{2}+\dfrac{4\,{\pi}^{2}\,\left({\varphi+n}\right)^{2}}{{l}^{2}}}}\]

    Рассмотрим уравнение:

    \[ p \left( \sqrt{p^2 + M^2} - b \right) = 0 \]

    Первая точка экстремума находится в:

    \[ p = 0 \]

    Остальные точки экстремума определяются из уравнения:

    \[ \sqrt{p^2 + M^2} - b = 0 \]

    Переносим \(b\) вправо:

    \[ \sqrt{p^2 + M^2} = b \]

    Поскольку \(b \in [-\infty, \infty)\), заметим, что корень не может быть отрицательным. Таким образом:

    \[ b \in [0, \infty) \]

    Находим точки экстремума:

    \[ p = \pm \sqrt{b^2 - M^2} \]

    Знак минус перед корнем можно отбросить, так как функция симметрична, и мы будем исследовать только область \(p \geq 0\).

    Важно также заметить, что корни существуют только при:

    \[ b^2 - M^2 > 0 \]

    С учётом этого условия:

    \[ b > M \]

    Подведем итог. Мы имеем две точки экстремума:

    • \(p = 0\)
    • \(p = \sqrt{b^2 - M^2}\), если \(b \in [M, \infty)\)
    Значения функции в точках экстремума:

    Найдём значения функции (1) в точках экстремума:

    1. Для \(p = 0\):

    \[ F(0, n, \mu, L, b, M, \phi) = \mu - \sqrt{(M - b)^2 + \frac{4\pi^2}{L^2}(n + \phi)^2} \]

    2. Для \(p = \sqrt{b^2 - M^2}\):

    \[ F(\sqrt{b^2 - M^2}, n, \mu, L, b, M, \phi) = \mu - \sqrt{\left(\sqrt{M^2 + b^2 - M^2} - b\right)^2 + \frac{4\pi^2}{L^2}(n + \phi)^2} \]

    Упрощая, получаем:

    \[ F(\sqrt{b^2 - M^2}, n, \mu, L, b, M, \phi) = \mu - \frac{2\pi}{L}\vert n + \phi \vert \]

    Найдем выражение для \(N_{max}\): Начнем со случая, если максимум будет в точке \(p=0\) \[ \mu - \sqrt{(M - b)^2 + \frac{4\pi^2}{L^2}(n + \phi)^2} > 0 \] т.е. нам интересны те случаи когда максимальное значение функции будет больше 0, чтобы было хоть что-то для интегрирования \[ \sqrt{(M - b)^2 + \frac{4\pi^2}{L^2}(n + \phi)^2} < \mu \] \[ (M - b)^2 + \frac{4\pi^2}{L^2}(n + \phi)^2 < \mu^2 \] \[ \frac{4\pi^2}{L^2}(n + \phi)^2 < \mu^2 - (M - b)^2 \] \[ \frac{2\pi}{L} \vert n + \phi \vert < \pm \sqrt{\mu^2 - (M - b)^2} \] Отметим, что \(\mu^2 - (M - b)^2 \geq 0 \)

    Также нужно обратить на знак минус справа, так как слева стоит модуль, и выражение положительно всегда, то справа не может быть отрицательного знака

    \[ \vert n + \phi \vert < \frac{L}{2\pi} \sqrt{\mu^2 - (M - b)^2} \] Теперь нужно раскрыть модуль, так \(\phi \neq 0\) и \(n\) может быть любого знака

    Если \(n + \phi \geq 0\)

    \[ n + \phi < \frac{L}{2\pi} \sqrt{\mu^2 - (M - b)^2} \] \[ n < \frac{L}{2\pi} \sqrt{\mu^2 - (M - b)^2} - \phi \] Т.е. \[ N_{max}(\mu, L, b, M, \phi) = \lfloor \frac{L}{2\pi} \sqrt{\mu^2 - (M - b)^2} - \phi \rfloor \]
